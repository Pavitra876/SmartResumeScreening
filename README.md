# Smart Resume Screening and Career Improvement System

AI-based resume analysis and career recommendation platform built with Streamlit, pypdf, and spaCy.

## Features
- Resume upload (PDF) and text extraction
- ATS score calculation
- Skill gap detection
- Personalized career recommendations
- Admin dashboard with analytics
- PDF report generation

## Tech Stack
Python · Streamlit · SQLite · pypdf · spaCy (NLP) · Pandas · NumPy · Matplotlib · Plotly

## Setup
\`\`\`bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python -m spacy download en_core_web_sm
streamlit run app.py
\`\`\`
